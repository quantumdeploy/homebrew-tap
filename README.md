# Homebrew Tap — QuantumLayer Platform

Homebrew formulae for [QuantumDeploy](https://quantumdeploy.ai) CLI tools.

## Install

```bash
brew install quantumdeploy/tap/qd
```

## Usage

```bash
qd auth login
qd analyse https://github.com/your-org/your-repo
qd generate <analysis-id> --cloud azure --output ./infra/
```
